# Knitpicking

A mobile-first Progressive Web App for tracking knitting and crochet projects.

**Live App:** https://knitpicking-vercel.vercel.app/

## Features

- Create multiple projects with custom names and colors
- Track two counters per project with customizable labels
- Data persists locally via localStorage
- Works offline as a PWA
- Installable to home screen on mobile devices
- Dark theme with pink accents

## Tech Stack

- React 18
- Material-UI v5
- Service Workers for offline support
- Deployed on Vercel

## Development

```bash
cd knitpicking-app
npm install
npm start
```

Build for production:

```bash
npm run build
```

## Installation as Mobile App

**iOS:** Safari → Share → Add to Home Screen

**Android:** Chrome → Menu → Add to Home Screen
